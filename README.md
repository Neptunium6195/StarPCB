# StarPCB

A simple double-layer LED star PCB (traces mostly on top, some on bottom) that will pulse correspondingly to the timer pulses. It has a button to initally power the system as well. The blinking works by connecting the LEDs to pin 3 of the timer which is connected to the capacitor which charges/discharges itself repeatedly, thus causing the timer to go on and off and ultimately the LEF's to turn on an doff. This was from Sleepover's April Challenge.
